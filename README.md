## 4. 영화 추천 알고리즘
- 사용 패키지
    - scikit-learn, pandas
- 영화 별 유사한 영화 추천 알고리즘
    - 해당 영화의 [제목, 장르, 키워드, 상영 언어, 제작 회사, 개요]의 비중을 조절하여 코사인 유사도를 계산
    - 각 카테고리의 비중을 조절하는 방식
        - 해당 카테고리를 XX배 증가하여 포함시키는 방식으로 비중을 조절하여 계산
        - 1순위 : 제목 -> 속편이나 시리즈를 가장 먼저 추천해주기 위함
        - 2순위 : 장르

- 사용자 별 북마크 기준 영화 추천 알고리즘
